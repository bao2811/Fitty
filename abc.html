Postman Mastery

Variables

  • • Environments: Dev, Staging, Prod.
  • • Global: Token, Base URLs.
  • • Dynamic: {{post_id}} từ response.

Scripts (JS)

  • • Pre-request: Hash mật khẩu, lấy giờ hệ thống.
  • • Tests: Assertion kết quả (pm.test).

Mocks & Docs

  • • Mock Server: Giả lập API cho Frontend.
  • • Auto-Docs: Xuất tài liệu Swagger-style.
pm.test("Dữ liệu trả về đúng định dạng", function () {
  let json = pm.response.json();
  pm.expect(json).to.have.property('id');
  pm.expect(json.status).to.eql('active');
});

Automation với Newman

Newman mang sức mạnh của Postman vào Command Line, cho phép tích hợp trực tiếp vào quy trình CI/CD.

Lợi ích:

  • Chạy tự động khi Developer đẩy code.
  • Không cần mở app Postman thủ công.
  • Xuất báo cáo HTML đẹp mắt.
# Cài đặt
npm install -g newman

# Chạy Collection
newman run api_test.json -e staging.json

# Với báo cáo HTML
newman run test.json -r htmlextra

"Tự động hóa giúp loại bỏ sai sót do con người."
